# Duty-game

This project is a **Turn-Based Strategy (TBS) game** available for both **PC and mobile**. Players can plan their moves, manage resources, and engage in strategic battles in a tactical map environment.  

**Note:** The game is currently **unfinished**. Balance is not yet complete, and only a **sample of units** is available for testing.

## Features

- Strategic, turn-based gameplay
- Tactical map with unit positioning
- Resource and unit management
- Local multiplayer
- Available on PC and mobile
